# HomeWeather

Проект показывает данные с датчиков NodeMCU в GNOME Shell через удалённый сервер.

## Структура проекта

- `backend/` — Node.js-сервер на Express, принимающий данные от NodeMCU и выдающий их GNOME-расширению.
- `gnome-extension/` — расширение для GNOME Shell, которое запрашивает данные с сервера и отображает их в панели.
- `nodeMCU/` — проект для платы NodeMCU / Arduino, собирающий данные датчиков и отправляющий их на сервер.

## Как это работает

1. NodeMCU считывает показания с датчиков (температура, CO2, влажность, давление).
2. Устройство отправляет данные на сервер через HTTP POST на маршрут `/set`.
3. Сервер сохраняет последние значения и возвращает их по запросу `/get`.
4. GNOME-расширение регулярно запрашивает `/get`, получает JSON с актуальными показаниями и отображает их в системной панели.

## Как запустить

1. Установите зависимости для бэкенда:
   ```bash
   cd backend
   npm install
   ```
2. Запустите сервер:
   ```bash
   npm start
   ```
3. Установите и активируйте GNOME-расширение:
   - Скопируйте папку `gnome-extension` в `~/.local/share/gnome-shell/extensions/home-weather@basher.com/`.
   - Перезапустите GNOME Shell (`Alt+F2`, затем `r`) или выполните `gnome-extensions restart home-weather@basher.com`.
